This document provides a detailed overview of the `testdriveform` component, which allows users to schedule test drives for Voltrux vehicles.

## Component Preview

### Live Preview

Here's a live preview of the Test Drive Form component (interactive version):

<TestDriveFormPreview />

The Test Drive Form component includes:
- A comprehensive form for user information
- Dropdown selections for location, model, and time
- Informational section highlighting benefits of test driving
- Responsive design that works on all devices
- Form validation and submission feedback

<!--/*

## CSS Styles
The component uses the following CSS classes:

### Main Container
- `.test-drive-container` - Main container with flex layout and responsive design
- `.test-drive-form-col` - Form column that takes 2/3 space
- `.test-drive-info-col` - Info column with neutral background and padding

### Form Header
- `.form-header h2` - Form title with XXL font size and bold weight
- `.form-header p` - Form subtitle with large font size and muted color

### Form Elements
- `.form-group` - Form field container with margin
- `.form-group label` - Label styling with bold weight
- `.form-group input` - Input field styling with border and padding
- `.form-group select` - Select dropdown styling with border and padding

### Form Buttons
- `.cta-button` - Submit button with accent background and hover effects
- `.cta-button:hover` - Hover effect with darker background
- `.cta-button:disabled` - Disabled state styling

### Messages
- `.form-message` - Form message container with padding and rounded corners
- `.form-message.success` - Success message styling with green background
- `.form-message.error` - Error message styling with red background
- `.error-details` - Error details styling with monospace font

### Info Boxes
- `.info-box` - Info section container with margin
- `.info-box h3` - Info section title with XL font size and border bottom
- `.info-box ul` - List styling with no bullets
- `.info-box ul li` - List item with checkmark icon
- `.why-voltrux ul li::before` - Checkmark icon for benefit list items

### Responsive Effects
- Media query for mobile screens changes layout to column
- Mobile-specific styling with info column moved to top
- Responsive form layout that adapts to screen size
