The Article List component supports interactive filtering and a responsive layout that adapts to different screen sizes. Here's a description of what you would see when the component is rendered:

### Desktop View
- A featured article is prominently displayed at the top in a two-column grid layout (image on one side, content on the other)
- Category filter buttons appear in a horizontal row below the featured article
- Remaining articles are displayed in a responsive grid format (typically 2-3 columns)

### Mobile View
- Featured article appears full-width with image above content
- Category filters become a scrollable horizontal row or stack vertically
- Article grid adjusts to 1 column for better readability on small screens

### Interactive Features
- **Category filtering**: Clicking on category buttons highlights the selected category and filters the displayed articles
- **Hover effects**: Article cards display a subtle shadow and border color change on hover
- **Image hover**: Article images slightly scale up when hovered
- **Responsive design**: Layout automatically adjusts based on screen size

### JavaScript Interaction

The component includes JavaScript functionality for category filtering:

```javascript
// Example JavaScript for category filtering
document.addEventListener('DOMContentLoaded', function() {
  const component = document.querySelector('[data-component="article-list"]');
  const filterButtons = component.querySelectorAll('.category-filter button');
  const articleCards = component.querySelectorAll('.article-card');

  filterButtons.forEach(button => {
    button.addEventListener('click', function() {
      const filterValue = this.getAttribute('data-filter');

      // Update active state
      filterButtons.forEach(btn => btn.classList.remove('active'));
      this.classList.add('active');

      // Filter articles
      articleCards.forEach(card => {
        if (filterValue === 'all' || card.getAttribute('data-category') === filterValue) {
          card.style.display = 'block';
        } else {
          card.style.display = 'none';
        }
      });
    });
  });
});
```
