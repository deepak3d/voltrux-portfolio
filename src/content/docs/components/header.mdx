## Overview

The Header component is a responsive navigation component for the Voltrux Motors website. It features a sticky positioning, brand logo, main navigation, call-to-action button, and mobile-friendly navigation with hamburger menu.

## Features

- Sticky header that remains at the top when scrolling
- Responsive design with desktop and mobile layouts
- Accessible navigation with focus states
- Mobile-friendly hamburger menu
- Scroll effect with enhanced shadow

## Component Structure

The Header component consists of several key CSS classes that define its functionality and appearance:


```html
<header class="header">
  <div class="header__container">
    <div class="header__logo">
      <img src="${model.logoPath @ context='uri'}" alt="${model.logoAltText}" class="header__logo-image" />

    <nav class="header__nav">

          <li>

          </li>
        </sly>
      </ul>
      <a href="${model.ctaLink @ context='uri'}" class="header__cta-button">${model.ctaText}</a>
    </nav>
    
    <button class="header__menu-toggle" aria-label="Toggle navigation menu">
      <span class="header__menu-icon"></span>
      <span class="header__menu-icon"></span>
      <span class="header__menu-icon"></span>
    </button>
  </div>
  
  <nav class="header__mobile-nav">
    <ul class="header__mobile-list">
      <sly data-sly-list.navItem="${model.navLinks}">
        <li>
          <a href="${navItem.url}" class="header__mobile-link">${navItem.text}</a>
        </li>
      </sly>
      <li>
        <a href="${model.ctaLink @ context='uri'}" class="header__cta-button--mobile">${model.ctaText}</a>
      </li>
    </ul>
  </nav>
</header>
```

## JavaScript Functionality

The header component includes JavaScript for enhanced functionality:

### Mobile Menu Toggle
- Toggles the mobile navigation menu when the hamburger button is clicked
- Updates the aria-expanded attribute for accessibility
- Adds/removes the `header__mobile-nav--active` class to show/hide the menu

### Scroll Effect
- Adds the `header--scrolled` class when the user scrolls more than 50px
- Enhances the shadow effect to indicate scroll position
- Improves visual hierarchy when scrolling

### Sample JavaScript Code
```javascript
/* header.js - Voltrux Motors Header Component */
document.addEventListener("DOMContentLoaded", () => {
  const toggle = document.getElementById("menu-toggle");
  const mobileNav = document.getElementById("mobile-nav");
  const header = document.querySelector(".header");

  if (toggle && mobileNav) {
    toggle.addEventListener("click", () => {
      mobileNav.classList.toggle("header__mobile-nav--active");
      toggle.classList.toggle("header__menu-toggle--active");
      toggle.setAttribute("aria-expanded", mobileNav.classList.contains("header__mobile-nav--active"));
    });
  }

  window.addEventListener("scroll", () => {
    header.classList.toggle("header--scrolled", window.scrollY > 50);
  });
});
```
